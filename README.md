# Telegram Mini App - Factions & Smart Chat Bot

## Features
- **Factions**: Ring 💍, Diamond list 💎, Alliance 🤝, Syndicate 💰, Order ⚔️ - Navigate to faction pages
- **Smart Chat Bot**: Full AI chat in `chat.html`
  - Faction info on query (e.g. "Ring?")
  - Greetings, app info, random replies
  - Telegram WebApp integrated (user name, sendData, theme)
- **Theme**: Premium gold/diamond design

## Run
- Frontend: `start index.html` or `start chat.html`
- Laravel PHP backend: `cd laravel-mini-app && php -S localhost:8000 -t public`

## Telegram Deployment
1. BotFather → Create bot → /mybots → Edit Bot → Menu Button → URL (your host)
2. Render.com for free deploy (see laravel-mini-app/README_LARAVEL.md)

Enjoy the smart bot! 🤖✨
