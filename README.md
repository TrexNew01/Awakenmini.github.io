# Simple Telegram Mini App

## Local Test
Open `index.html` in browser.

## Deploy to Vercel (for Telegram Bot)
1. Install Vercel CLI: `npm i -g vercel`
2. Login: `vercel login`
3. Deploy: `vercel` (in this dir)
4. Get URL (e.g., https://your-app.vercel.app)
5. @BotFather → your bot → /setmenubutton → enable web app → paste URL
6. Open bot in Telegram, tap button.

App greets user, test button shows alert.
